## Seven-Chakra Framing

The first awakening loop should be explained as a seven-chakra practice, not only as a streak counter.

Suggested sequence:

1. Root chakra - red - grounding
2. Sacral chakra - orange - creativity
3. Solar plexus - yellow - confidence
4. Heart chakra - green - compassion
5. Throat chakra - blue - expression
6. Third eye - indigo - intuition
7. Crown chakra - violet / white - awakening

This gives users a reason to return beyond raw progression.

## Best Habit Loop

The healthiest daily loop is:

1. Open app
2. See one clear recommended action
3. Finish one short meditation
4. Watch orb growth advance
5. Claim or progress a mission
6. Occasionally store or bless an orb

This loop works because it combines:
- calm ritual
- visible progress
- collection
- social expression

## Beginner Journey Recommendation

New users should be guided through this order:

1. First meditation
2. Learn that each day corresponds to one chakra
3. Grow first orb layer
4. Open garden
5. Understand that awakening takes 7 chakra days
6. Store one orb
7. Send one blessing
8. Return next day to continue awakening streak

## Mission Design Directions

### Good mission families
- Ritual: meditate at dawn, at night, or on consecutive days
- Collection: store orbs, unlock forms, awaken an orb
- Exploration: try different meditation categories
- Social: send blessings, receive blessings
- Mastery: maintain streaks, finish longer sessions

### Avoid
- pure grind with no meaning
- too many missions unlocking at once
- missions that do not teach a feature

## UI Guidance To Build Better Habits

### 1. Home should show one primary CTA
Recommended card:
- "Today's next step"
- Example: "Meditate 7 minutes to grow your orb"

### 2. Show today's orb growth bar
Need a compact indicator:
- Day X of 7
- Minutes completed today
- Minutes remaining today

### 3. Show locked content with reasons
For every locked audio / shape:
- explain why it is locked
- show exactly how to unlock it
- provide a button to go there

### 4. Use mission CTA buttons everywhere
Each task should say:
- Go Meditate
- Go Garden
- Go Upgrade

### 5. Add a gentle onboarding banner for new users
Recommended content:
- "Complete these 5 steps to understand PSI-G"

### 6. Add a daily return reason
Examples:
- continue awakening streak
- unlock next orb layer
- complete one beginner task
- receive today's spiritual guidance
